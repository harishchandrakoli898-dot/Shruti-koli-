# Suyash Classes Website

A modern, responsive website for Suyash Classes - an educational institution dedicated to providing quality education and improving student performance.

**Tagline:** "Don't think about past, think about future."

## 🎯 About

Suyash Classes is an educational platform offering quality teaching with a focus on student success. This website showcases our facilities, services, and mission to help students achieve their academic goals.

## ✨ Features

- **Hero Section** - Eye-catching landing page with call-to-action
- **Facilities Showcase** - Highlights key offerings:
  - Best Education with dedicated teachers and modern teaching methods
  - Well-Equipped Classrooms with smart learning tools
  - Focus on Student Performance through continuous evaluation and personalized guidance
- **Responsive Design** - Mobile-friendly and accessible interface
- **Clean UI** - Professional and modern appearance

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic HTML/CSS/JavaScript knowledge (for development)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/harishchandrakoli898-dot/Shruti-koli-.git
cd Shruti-koli-
```

2. Open the project in your browser:
```bash
# Simply open index.html in your browser
open index.html  # macOS
start index.html # Windows
```

Or use a local server:
```bash
# Python 3
python -m http.server 8000

# Then visit http://localhost:8000
```

## 📝 Usage

The website includes:
- A hero section with a prominent call-to-action ("Join Us Today")
- A facilities section showcasing three key features
- Responsive card-based layout for easy navigation

To customize:
- Update the hero text and button link in the HTML
- Modify the facilities information as needed
- Adjust colors and styling in the CSS file

## 🎨 Customization

To personalize the website:

1. **Change the tagline** - Edit the text in the hero section
2. **Update facilities** - Add or modify facility cards
3. **Add images** - Include pictures in each section
4. **Update links** - Connect the "Join Us Today" button to your registration page
5. **Add more sections** - Consider adding: Courses, Testimonials, Contact Form, Faculty Details
